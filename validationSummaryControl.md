<!-- ValidationSummary Control in ASP.NET -->

<!-- The ValidationSummary control is used to display a summary of all validation errors that have occurred in a Web page in a single place. -->

<!-- In general, in a real-time application, it is common to display a red star symbol next to the input control where the error occured, and then the detailed error message in the validation summary control. -->

<!-- Properties specific to the validation summary control: -->

<!-- 1. HeaderText -->
<!-- Changes the header text. -->

<!-- 2. ShowSummary -->
<!-- Determinjes whether to display the summary text of all the validation errors. -->

<!-- 3. ShowMessageBox -->
<!-- Determines whether to display a mesage box with all the validation errors. -->

<!-- 4. DisplayMode -->
<!-- Determines the display format of the summary. It can be set to List, BulletList, or SingleParagraph. -->
